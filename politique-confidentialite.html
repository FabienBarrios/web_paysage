2. Données collectées

2.1 Données collectées directement

TailleZone ne collecte des données personnelles que si vous nous les transmettez volontairement via :

  • Le formulaire de contact (si vous nous contactez pour un devis ou une demande d'information)

2.2 Quelles données ?

Les données que nous pouvons collecter incluent :

  • Identité : Nom, prénom
  • Contact : Email, téléphone, adresse postale
  • Demande : Nature de votre projet, message

Important : Le site taillezone.fr n'utilise AUCUN cookie de tracking (Google Analytics, Facebook Pixel, etc.). Aucune donnée n'est collectée automatiquement lors de votre navigation.

4. Destinataires des données

Vos données personnelles sont destinées exclusivement à :

  • TailleZone (personnel habilité)
  • Hébergeur AWS (Amazon Web Services) - uniquement pour l'hébergement technique du site

Nous ne vendons, ne louons et ne partageons jamais vos données avec des tiers à des fins commerciales.

5. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Obtenir une copie de vos données personnelles
  • Droit de rectification : Corriger des données inexactes
  • Droit à l'effacement ("droit à l'oubli") : Supprimer vos données dans certains cas
  • Droit d'opposition : Vous opposer au traitement de vos données
  • Droit à la limitation du traitement : Demander la suspension du traitement
  • Droit à la portabilité : Récupérer vos données dans un format structuré
  • Droit de définir des directives post-mortem : Définir le sort de vos données après votre décès

Comment exercer vos droits ?

Pour exercer vos droits, contactez-nous :

  • Par email : taille-zone@gmail.com (en précisant votre demande)
  • Par téléphone : 06 61 20 48 16
  • Par courrier : TailleZone, [Votre adresse]

Délai de réponse : Nous nous engageons à répondre dans un délai d'1 mois maximum.

6. Sécurité des données

TailleZone met en œuvre des mesures techniques et organisationnelles appropriées pour protéger vos données personnelles contre :

  • L'accès non autorisé
  • La divulgation
  • La modification
  • La destruction

Mesures de sécurité :

  • Hébergement sécurisé sur AWS (certifié ISO 27001)
  • Transmission des données via HTTPS (certificat SSL/TLS)
  • Accès limité aux données (uniquement personnel habilité)
  • Pas de stockage de données bancaires sur notre site

7. Cookies

Le site taillezone.fr n'utilise aucun cookie de tracking, publicitaire ou analytique.

Les seuls cookies pouvant être utilisés sont des cookies techniques strictement nécessaires au fonctionnement du site (session, sécurité). Ces cookies ne nécessitent pas votre consentement selon les directives de la CNIL.

8. Transfert de données hors UE

Vos données personnelles sont hébergées au sein de l'Union Européenne (Luxembourg) via Amazon Web Services. Aucun transfert de données hors UE n'est effectué.

En cas de modification substantielle, nous vous en informerons par email ou via un message sur le site.
